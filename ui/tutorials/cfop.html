What is CFOP?

CFOP stands for Cross, F2L, OLL, PLL. It is also sometimes called the Fridrich method because it was popularized by Jessica Fridrich. However, some cubers say it is better to use a term like CFOP because it is more descriptive of the method.

Each letter in CFOP stands for one of the steps, so as you can probably guess, CFOP is broken down into 4 steps:

  1. Cross
  2. First 2 layers (F2L)
  3. Orient last layer (OLL)
  4. Permute last layer (PLL)

Why use CFOP?

The goal of CFOP is to solve the cube faster than the beginner method.

How does the CFOP method help with solving the Rubik's Cube faster? The main benefit is that it takes fewer moves than the beginner method. If you are familiar with the beginner method, you probably know that it includes 7 steps:

  1. Cross
  2. First layer
  3. Second layer
  4. Orient edges
  5. Orient corners
  6. Permute corners
  7. Permute edges

CFOP takes fewer moves than the beginner method because it combines steps:

  • First layer and second layer ➡ First 2 layers
  • Orient edges and orient corners ➡ Orient last layer
  • Permute corners and permute edges ➡ Permute last layer

You should already know how to solve the cross, so now we will jump straight into learning F2L!

F2L

F2L stands for First 2 Layers, and as the name implies, the goal is to solve the first 2 layers of the Rubik's Cube in one step.

When you are starting out with F2L, you want to learn the cases intuitively. That means you should understand how the pieces move instead of memorizing algorithms. As you become more advanced, you will start to ingrain the cases in your muscle memory, and you will be able to solve F2L without thinking deeply.

OLL

OLL stands for Orient Last Layer, and the goal is to orient the last layer of the Rubik's Cube. For example, if you started with the white cross, then the goal is to get all of the yellow stickers on top.

This step is algorithmic, which means you just have to memorize algorithms. There are 57 algorithms for full OLL, but many of them are just mirrors or inverses of each other.

This is what OLL should look like when it's done.

PLL

PLL stands for Permute Last Layer, and much like OLL, this step is also algorithmic. There are 21 algorithms for full PLL.

After PLL, the entire cube should be finished!
